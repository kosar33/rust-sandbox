# rust-sandbox 🦀

Легковесный локальный веб-интерфейс и автоматизированный менеджер запуска для Rust-приложений (включая Axum с поддержкой `edition = "2024"`), оптимизированный для мобильных устройств (через Termux на Android) и ПК.

---

## ⚡ Быстрая установка в 1 строку (Android / Termux)

Для моментального развертывания проекта со всеми зависимостями скопируйте в Termux следующую команду:

```bash
pkg install curl -y && curl -# -L https://raw.githubusercontent.com/kosar33/rust-sandbox/main/install.sh -o install.sh && bash install.sh && rm install.sh
```

*Если загрузка установщика через curl не начинается, у вас нет доступа к серверам github по каким либо причинам 🌝.*

---

## 🌐 Офлайн режим и интернет-зависимости

Для работы этого проекта доступ к интернету требуется **исключительно на этапе установки**.

### Куда подключается установщик:
*   **Termux mirrors:** Системные пакеты (`nodejs`, `rust` и др.).
*   **NPM registry (`npmjs.org`):** Модули Express и WebSocket.
*   **Crates.io:** Зависимости Rust (`axum`, `tokio` и др.). Скрипт принудительно скачивает и компилирует их заранее.
*   **Tailwind & Prism CDN:** Копирует JS/CSS стили и подсветку во внутреннюю папку `public/`.

После завершения установки вы можете полностью выключить интернет. Проект работает 100% локально. **Доступ к сети потребуется только в том случае, если вы решите добавить новые библиотеки (crate) в `Cargo.toml`.**

## 🚀 Возможности

*   **Офлайн-подсветка синтаксиса:** Встроенная подсветка кода (Rust, TOML) на базе локальной Prism.js [1.1], оптимизированная для мобильных устройств.
*   **Локальные стили:** Интерфейс автономен благодаря локальному компилятору Tailwind.
*   **Интеграция с ИИ (Импорт в 1 клик):** Автоматический разбор единых шаблонов от нейросетей.
*   **Шаблон промпта:** Встроенная кнопка копирования структуры запроса для ИИ.
*   **Терминал реального времени:** Вывод процесса компиляции прямо на экран через WebSocket.
*   **Адаптивная верстка:** Корректное отображение на мобильных экранах под навигационной панелью браузера (использует `h-dvh`).

## 🛠 Запуск

Перейдите в папку проекта и запустите сервер:
```bash
cd rust-sandbox # (или имя папки, которое выдал скрипт)
node server.js
```
Сверните Termux, откройте браузер на смартфоне и перейдите на:
👉 `http://127.0.0.1:3000/`
